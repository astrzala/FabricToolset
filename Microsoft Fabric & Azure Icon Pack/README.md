# Microsoft Fabric Architecture Icons 🧩

A curated set of **Microsoft Fabric and Azure icons optimized for architecture diagrams**.

The icons were extracted from the Microsoft Fabric UI and organized into **clean, ready-to-use SVG files and Draw.io libraries**.

## Features

- Lossless **SVG format**
- **Uniform 60px height** for consistent diagrams
- **Consistent border style**
- **Internal padding removed** for better alignment
- **Ready-to-use Draw.io libraries**

## 🚀 Using the Icons

### 🌐 Draw.io Online

Open the library link and it should load automatically:

**Microsoft Fabric Draw.io Icon Pack**

If the libraries do not load automatically:

1. Open **draw.io**
2. Go to **Configuration**
3. Import the library file

### 💻 Draw.io Desktop

1. Download the `.drawio` library files from this repository  
2. Open **draw.io**  
3. Go to **File → Open Library**  
4. Select the downloaded library
